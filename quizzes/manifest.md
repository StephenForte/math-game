# Quiz Manifest

This file controls which quizzes appear in Math Practice Lab and the order they
show up in. To add a quiz, create a new `.md` file in this `quizzes/` folder and
add its filename to the `quizzes` list below. To remove one, delete its line.

No code changes or rebuilds are needed — just edit Markdown and reload the page.

```json
{
  "quizzes": [
    "multiplication.md",
    "fluency.md",
    "fractions.md",
    "simplify-fractions.md",
    "decimals.md",
    "geometry.md"
  ]
}
```
